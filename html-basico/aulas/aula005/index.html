Algumas tags de formatação de texto

Tag <b> (não semântica)

Esta é uma tag para deixar o texto em negrito, mas a tag <b> não é uma tag semantica (de melhor aproveitamento pelos mecanismos de busca).

Tag <strong> (semântica)

A tag <strong> é uma tag que também deixa seu texto em negrito, mas como se diz de uma forma semântica, dixa seu texto mais pesado, e é uma tag semântica, com isso é melhor aproveitada pelos mecanismos de busca

Tag <i> (nâo semântica)

A tag <i> é uma tag que deixa seu texto em itálico, mas é uma tag não semântica e com isso é meno aproveitada pelos mecanismos de busca.

Tag <em> (semântica)

Já a tag <em> é uma tag que dá ênfase no seu texto (deixando-o em italico) mas de uma forma semântica, ou seja, melho aproveitada pelos mecanismos de busca.

Tag <mark> (semântica)

Esta tag faz uma marcação em seu texto, que por padão é feita na cor amarela.

Tag <small> (semântica)

Esta tag faz com que seu texto fique um pouco menor que o resto da linha!

tag <s> (não semântica)

Esta tag irá riscar um texto como se ele estivesse sido removido, mas de uma forma não semântica.

Tag <del> (semântica)

Esta tag irá riscar um texto como se ele estivese sido removido!

Tag <u> (não semântica)

Esta tag deixa o texto sublinhado, normalmente como se tivesse sindo inserido, mas é uma tag não semântica, com isso pouco aproveitada pelos mecanismos de busca.

Tag <ins> (semântica)

Esta tag deixa o texto sublinhado, normalmente como se tivesse sindo inserido, e essa tag é semantica, por isso é muito aproveitada pelos mecanismos de busca.

Tag <sub> (semântica)

Esta tag deixa seu texto subscript, tipo H2O e por ser uma tag semântica é bem aproveitada pelos mecanismos de busca.

Tag <sup> (semântica)

Esta tag deixa seu texto superscript, tipo X2 = 10, e por ser uma tag semântica pe bem aproveitada pelos mecanismos de busca.

OBS:

Ao utilizar as tags de formatação de textos, sempre utilize as tags SEMÂNTICAS, pois as não semânticas estão em desuso!
