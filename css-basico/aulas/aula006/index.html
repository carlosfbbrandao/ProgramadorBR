Cores

Cores pelo nome

Podemos trabalhar com cores chamando-as pelo nome. O HTML já carrega várias cores com o nome.

Cores pelo RGB

Podemos trabalhar com as cores em RGB, que recebera tre valores referentes a red, greem, blue, que irão formar as outras cores.

Cores pelo RGBA

A diferença entre o RGB e o RGBA, é que o ultimo valor se refere a transparência.

Cores pelo exadecimal

As cores exadecimais usam um código numérico com o simbolo # antes.
